# umplus

E-wallet สำหรับโรงเรียนที่ถูกพัฒนาขึ้นเพื่อ 
ความสะดวกสะบายในการใช่จ่ายเงินใน โรงเรียน 

โดย โครงงานตัวนี้ถูก พัฒนาขึ้นมาอย่างเร่งรีบภายในเวลา 1 เดือน อาจทำให้มี bugs บางส่วนที่ยังไม่ได้ถูกแก้ไข
